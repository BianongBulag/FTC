====================================================
WHAT'S HURTING THE SCORE
====================================================

CRITICAL
--------
1. Every page uses the same title and description
   - All pages show: "Feed the Curious | Food, Coffee, Culture & Stories"
   - Google penalizes duplicate meta titles across pages
   - Fix: Add unique `export const metadata` to each page.tsx

2. No Open Graph tags
   - Social shares (Facebook, Messenger, Viber) show no image or title
   - Fix: Add og:title, og:description, og:image to each page metadata

3. No sitemap.xml
   - Google cannot efficiently discover all posts and pages
   - Fix: Create src/app/sitemap.ts (auto-generates from posts data)

4. Article hero uses CSS backgroundImage
   - Not optimized for LCP (Largest Contentful Paint)
   - Google PageSpeed flags this
   - Fix: Replace with Next.js <Image> component

5. No JSON-LD structured data
   - Missing rich results in Google (stars, author, date under search result)
   - Fix: Add BlogPosting schema to each article page

MINOR
-----
6. No robots.txt
7. No <link rel="canonical"> on pages
8. No Twitter/X card meta tags
9. No favicon defined in Next.js metadata API
